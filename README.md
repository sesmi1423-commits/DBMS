Overview

This is a simple To-Do List Manager built in Python using Google Colab.
It is a beginner-friendly project that demonstrates the use of:

Lists and dictionaries

Functions

Loops and conditionals

Basic user input/output

The project provides a text-based interface to help manage daily tasks in an interactive way.

🎯 Features

➕ Add a task – Create new tasks to track.

👀 View tasks – See all your tasks with status.

✅ Mark task as done – Update tasks when completed.

🗑️ Delete a task – Remove tasks from the list.

🚪 Exit – Quit the program safely.
his project helps beginners practice:

Python basics (variables, loops, conditionals)

Data structures (lists & dictionaries)

Functions & modular coding

User interaction with input/output
